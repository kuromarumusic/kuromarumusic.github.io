Experimente mit prozeduralen Inhalten

Meine Faszination für verschiedene Kunstformen trieb mich dazu mit einer Mischung aus Kunst und Programmieren zu experimentieren. Oft experimentiere ich mit Perlin oder Simplex Noise und auch mit Computererzeugter Musik.



Beispiel: Computergenerierte Musik & Visuals




Die Musik in diesem Video im Grunde durch einen eigenen und im Grunde simplen Algorithmus erzeugt:

  1. Aus einer Liste von Tonleitern wird eine Tonleiter wie zum Beispiel «Pentatonisch Moll» ausgewählt
  2. Aus dieser Tonleiter werden eine zufällige Anzahl Noten rausgepickt
  3. Jede Note bekommt eine zufällige Spieldauer, so wird ein Rhythmus erzeugt
  4. Verschiedene Drum-Rhythmen werden vordefiniert und davon wird zufällig einer gewählt.

Diese Methode kann relativ einfach «zufällige» Musik erzeugen, jedoch kann es sehr einfach zu willkürlich klingen und die Melodie keinen roten Faden haben. Dies würde sich z.B. durch Machine Learning oder einen statistischen Ansatz lösen lassen.
